# RTL Combinational Circuits in Verilog

This repository contains RTL implementations and verification testbenches for commonly used combinational logic circuits. The purpose of this project is to strengthen digital design fundamentals, RTL coding skills, and simulation-based verification.

## Implemented Designs

### Arithmetic Circuits
- Half Adder
- Full Adder
- Half Subtractor
- Full Subtractor
- Ripple Carry Adder
- Carry Lookahead Adder
- Adder/Subtractor

### Multiplexing Circuits
- 2:1 Multiplexer
- 4:1 Multiplexer
- 8:1 Multiplexer
- 16:1 Multiplexer
- Demultiplexer

### Encoders & Decoders
- Encoder
- Priority Encoder
- Decoder

### Comparators
- 1-bit Comparator
- 4-bit Comparator
- Magnitude Comparator

### Code Converters
- Binary to Gray Converter
- Gray to Binary Converter
- BCD to Excess-3 Converter
- Excess-3 to BCD Converter

### Other Combinational Blocks
- Barrel Shifter
- Arithmetic Logic Unit (ALU)
- Parity Generator
- Parity Checker

## Verification

Each design includes:
- RTL implementation in Verilog
- Dedicated testbench
- Functional verification through simulation
- Waveform analysis for output validation

## Tools Used

- Verilog HDL
- ModelSim / QuestaSim
- EDA Playground
- GTKWave

## Learning Objectives

- Develop RTL design skills
- Understand combinational circuit design
- Learn Verilog coding practices
- Gain experience writing testbenches
- Build a foundation for sequential logic and processor design
